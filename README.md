# 📊 Instagram Performance Analysis using SQL

## 📌 Project Overview
This project analyzes Instagram content performance to identify key engagement drivers and optimize content strategy. The analysis focuses on metrics such as reach, impressions, likes, comments, and shares.

---

## 🎯 Objectives
- Analyze performance across different post types (Reels, Images, Carousel, Videos)
- Identify key factors influencing engagement
- Understand audience behavior based on time and content category
- Provide actionable insights for content strategy improvement

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- PowerPoint (for presentation)

---

## 📂 Project Files
- `Instagram_SQL_Analysis.sql pptx` → Contains all SQL queries
- output_screenshorts → Contains all Sql Query Output
- `Instagram_Presentation.pptx` → https://youtu.be/zykA-AX-KOE  (Business presentation with insights)
- `README.md` → Project documentation

---

## 📊 Key Insights
- 📈 Reels generate the highest reach, impressions, and shares
- 📉 Carousel posts show comparatively lower performance
- ⏰ Posting time and day significantly impact engagement
- 🎯 Certain content categories consistently perform better

---

## 💡 Recommendations
- Focus more on high-performing formats like Reels
- Optimize or rethink low-performing formats like Carousel
- Plan content based on audience engagement trends
- Use data-driven strategies for continuous growth

---


## 🔗 Project Outcome
This project demonstrates how data analysis can help improve social media strategy and drive better engagement using data-driven decision-making.
